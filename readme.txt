La página consiste en un blog para subir partituras musicales, para hacerlo necesariamente la persona debe estar logueada:

Algunos detalles:
Para editar o borrar publicaciones se debe ingresar con un usuario y contraseña de administrador.
Todos los botones de la página son funcionales
Para publicar, ver la lista de publicaciones y ver perfil de usuario se debe estar logueado.
Para ver la lista de miembros, propósito de la página y acerca de mí, no hace falta estar logueado.
Todos los botones de redes sociales del footer redirigen al link de github en donde está subido el repositorio.
Para la búsqueda de partituras en la base de datos no hace falta estar logueado.
